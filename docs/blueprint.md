# **App Name**: Mail Pilot

## Core Features:

- Excel Data Import: Import data from an Excel file, parsing columns for recipient information, appointment details, and email content.
- Email Composition: Compose personalized emails using data from the Excel file. Support variables for names, dates, and other relevant details.
- Email Sending: Send personalized emails to recipients. Allow configuration for mail client such as authenticated SMTP.
- Email preview and testing: Generate an email preview from data selected within the Excel file
- Confirmation Message Generation: Leverage an LLM to dynamically generate and adapt email content, to avoid it sounding like a template. The LLM will act as a tool to provide wording suggestions for confirmation emails based on event type.

## Style Guidelines:

- Primary color: Deep Indigo (#3F51B5) to convey professionalism and reliability.
- Background color: Very light grey (#F5F5F5) to provide a clean, neutral backdrop.
- Accent color: Soft teal (#4DB6AC) for interactive elements, providing a touch of modernity and calmness.
- Body and headline font: 'Inter', a grotesque-style sans-serif font known for its clean and readable design, suitable for both headlines and body text.
- Use simple, professional icons to represent different functions and data types.
- Design a clean, intuitive interface with clear sections for data import, email composition, and sending options.
- Incorporate subtle animations for loading states and confirmations to improve user experience.
